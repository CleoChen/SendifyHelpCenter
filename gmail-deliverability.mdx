Gmail 拥有严格且智能的垃圾邮件过滤系统。新的发信域名和 IP 地址如果没有经过适当预热，邮件可能进入垃圾箱或被拒收。本指南提供一套可执行的 Gmail 域名预热策略。

更多要求可参考 Google 官方的[电子邮件发件人指南](https://support.google.com/a/answer/81126)。

## 第一步：完成技术基础配置

发送第一封邮件前，必须完成发件人身份认证。这相当于邮件的数字身份证，是获得 Gmail 信任的基础。

### 配置 SPF、DKIM 和 DMARC

- **SPF（Sender Policy Framework）**：在 DNS 中声明哪些 IP 地址有权代表您的域名发送邮件，防止他人伪造域名。
- **DKIM（DomainKeys Identified Mail）**：为邮件添加数字签名，证明邮件由您的域名发出且在传输过程中未被篡改。
- **DMARC（Domain-based Message Authentication, Reporting, and Conformance）**：规定 Gmail 应如何处理未通过 SPF 或 DKIM 验证的邮件，并提供域名冒用报告。

<Note>
  预热初期建议将 DMARC 策略设置为 `p=none`，先观察和收集数据，避免直接拒绝可能存在配置问题的合法邮件。
</Note>

关于配置方法，请参阅[添加并验证发信域名](/sending-domain)。

### 配置 Google Postmaster Tools

访问 [Google Postmaster Tools](https://postmaster.google.com/)，按照页面提示验证发信域名。

配置完成后，可以查看：

- 域名信誉与 IP 信誉
- SPF、DKIM 和 DMARC 通过率
- 投递错误率
- 垃圾邮件投诉率

这些数据可以帮助您判断预热是否有效，并在出现问题时定位原因。

## 第二步：逐步预热域名

预热的核心是通过小批量、高质量、高参与度的发送，逐步向 Gmail 证明您是负责任且受欢迎的发件人。

### 筛选高质量种子用户

- 优先选择最近 30 天内登录、购买或明确同意接收邮件的活跃用户。
- 打开、点击以及将邮件标记为“不是垃圾邮件”等积极行为，有助于提升域名信誉。
- 不要使用购买的、陈旧的或未经确认的邮件列表进行预热。

### 制定循序渐进的计划

从较低发送量开始，每天或每隔几天以稳定的增长率增加发送量，直至达到目标日发送量。整个过程通常需要 1–2 周或更长。

以下计划仅供参考，需要根据实际数据调整：

| 阶段 | 建议发送量与操作 |
| --- | --- |
| 第 1–3 天 | 每天向 200–1,000 个最活跃的 Gmail 用户发送，密切观察 Postmaster Tools 和发送数据 |
| 第 4–7 天 | 如果打开率较高且投诉率接近零，逐步增加，例如 200 → 400 → 800 封 |
| 第 8–14 天 | 保持健康增长，例如每天 1,500 → 3,000 → 6,000 封 |

<Warning>
  如果垃圾邮件投诉率上升或打开率骤降，应立即暂停增加发送量，检查邮件内容和联系人质量。数据恢复后再继续增长。
</Warning>

### 保持稳定的发送频率

预热和日常发送期间都应保持相对稳定的发送模式。避免从每天 100 封突然增加到 10 万封，这类流量尖峰很容易被 Gmail 判定为可疑行为。

## 第三步：持续监控和优化

### 监控核心指标

- **垃圾邮件投诉率**：目标保持在 **0.3% 以下**，即每 1,000 封邮件少于 3 个投诉。一旦升高，应立即采取行动。
- **硬退信率**：目标保持在 **0.5% 以下**。较高的硬退信率通常表示联系人列表质量较差，需要清理。
- **打开率与点击率**：代表用户参与度。预热期间较高的参与度有助于快速改善信誉。

### 优化邮件内容

- 每封邮件都应提供清晰、易于点击的退订链接。让不感兴趣的用户退订，远好于被标记为垃圾邮件。
- 提供收件人期待且有价值的内容，并适当个性化，例如使用收件人姓名。
- 避免过多大写字母、惊叹号、夸张营销词汇和严重失衡的图文比例。

关于退订功能，请参阅[联系人退订](/unsubscribe)。

## 发送前检查清单

- 域名已正确配置 SPF、DKIM 和 DMARC（预热初期为 `p=none`）
- 已在 Google Postmaster Tools 中验证域名，并可以查看信誉数据
- 已筛选高质量、高活跃度的 Gmail 地址作为初始预热对象
- 已制定从低到高、循序渐进的发送计划
- 可以监控投诉率、退信率等指标，并在数据恶化时暂停计划
- 邮件内容提供明确价值和便捷的退订方式

耐心是 Gmail 域名预热的关键。保持稳定发送并持续改善联系人和内容质量，才能与 Gmail 建立长期信任关系。
