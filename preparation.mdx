开始发送邮件前，需要依次添加发信域名、创建发信地址并添加回信地址。

## 1. 添加发信域名

**功能入口：**「系统管理 → 域名管理」

添加并验证域名后，即可使用 `@你的域名` 的地址发信。建议使用自有域名。

如需了解两种发信方式的区别和完整 DNS 配置，请参阅[添加并验证发信域名](/sending-domain)。

### Step 1：输入发信域名

登录管理员账号，进入域名管理页面，点击「添加发信域名」。为了避免 SPF、MX 记录冲突，建议在企业邮箱域名下创建二级域名，作为营销邮件的发信域名。

![添加发信域名](/add-domain.png)

### Step 2：设置域名解析

根据系统指引，前往域名服务商的管理后台完成解析配置，然后点击「已配置完成」。

![配置域名解析](/configure-dns.png)

### Step 3：等待验证结果

系统会通过网络协议自动验证各项记录。配置生效通常需要几分钟至几小时。

![查看域名验证状态](/domain-status.png)

## 2. 创建发信地址

**功能入口：**「系统管理 → 发信地址」

发信地址可以分配给企业成员，成员随后即可使用获分配的地址发送邮件。

1. 完成域名设置后，进入发信地址管理页面。
2. 点击「新建发信地址」。
3. 填写发信地址，并可同时设置发信人昵称。
4. 点击「确定」，在发信地址列表中查看新地址。

![进入发信地址页面](/sender-address-entry.png)

![新建发信地址](/create-sender-address.png)

![查看发信地址列表](/sender-address-list.png)

## 3. 添加回信地址

**功能入口：**「系统管理 → 回信地址」

回信地址用于接收收件人对邮件的回复。

1. 点击右上角「添加回信地址」。
2. 输入回信地址并点击「确定」。
3. 前往对应邮箱，打开「回信地址验证邮件」。
4. 点击邮件中的「验证」按钮。
5. 返回回信地址页面查看新添加的地址。
