发信任务开始执行后，即可查看邮件发送情况，了解当前营销任务的效果。

<Icon icon="arrow-right-to-bracket" iconType="solid" /> `发信任务` → `查看详情`

## 查看数据

页面会展示邮件发送各节点的数据概览，以及每个节点下的收件人详情。

| 指标 | 说明 |
| --- | --- |
| 营销人数 | 创建任务时选定的联系人数量，即已选联系人减去例外联系人 |
| 发送封数 | 成功发出的邮件总数，排除额度限制、无效地址等未发出邮件；邮件额度按此数量消耗 |
| 送达封数 | 成功送达到对方邮箱的邮件数量 |
| 送达率 | 送达封数 ÷ 发送封数 × 100% |
| 打开封数 | 发出后状态为已读的邮件数量 |
| 打开率 | 打开邮件的用户数 ÷ 发送邮件总数 × 100% |
| 点击人数 | 点击邮件中链接的用户数 |
| 点击率 | 点击邮件中链接的用户数 ÷ 发送邮件总数 × 100% |

### 收件人详情

- **营销人数、已发送、已送达**：可查看收件邮箱地址、名、姓、所属标签和发信状态。
- **已打开**：除上述信息外，还可查看打开次数和最近打开时间。
- **已点击**：除上述信息外，还可查看点击次数和最近点击时间。

## 批量管理

- **一键打标**：根据收件人的打开、点击和退订行为，快速标记不同用户类别。
- **便捷导出**：导出高意向联系人的邮箱地址，方便后续跟进。
